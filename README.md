## Projektübersicht

Das Ziel dieses Projekts ist es, das Thema Freiheit aus verschiedenen Blickwinkeln verständlich zu erklären. Jede Seite des Memoflips beantwortet eine bestimmte Fragestellung aus dem Unterricht und wird durch eine passende Comic-Illustration visuell unterstützt.
